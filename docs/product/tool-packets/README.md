# GrowPathAI Tool System Packets

These packets are implementation handoffs for the real GrowPathAI tool system.

Core rule:

```txt
No toy calculators.
No fake Open buttons.
No planned user-facing tools pretending to work.
Every built tool must save/reload, connect to grow context, create logs/tasks/timeline events where relevant, and preserve uncertainty.
```
